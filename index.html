Totó Stats BETA

  • SofaScore não fornece gráficos para todas as partidas.
  • Placares/Estatísticas são atualizados automaticamente.
  • Use zoom para melhor visualização (CTRL + Scroll).
  • Arraste/Selecione um cartão de partida clicando e segurando no fundo dele.

Totó Stats é um projeto que usa a API do SofaScore para fornecer placares ao vivo e estatísticas de partidas de futebol.
